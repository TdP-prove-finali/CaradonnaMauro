Istruzioni per l'installazione e l'utilizzo dell'applicazione:
- fare il fork del progetto in GitHub e clonarlo sul proprio editor (es. Eclipse)
- popolare il proprio database con i dati presenti nel progetto e modificare la password per accedere al database 
  nella classe DBConnect nel package it.polito.tdp.acquisti.dao
- aprire il progetto e fare Run della classe Main
